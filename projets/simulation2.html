Optimisation de la réduction du bruit dans une salle

Ce projet a été réalisé durant ma deuxième année de master, dans le cadre de l'Unité d'Enseignement "Simulation et Copules". J'ai mené ce projet en collaboration avec Shuaib Mohun. L'objectif est de minimiser l'impact sonore d'une salle en réorganisant les élèves selon des critères définis. Nous avons pour cela exploré plusieurs algorithmes : le Recuit Simulé Basique, le Recuit Simulé Ciblé, le Recuit Simulé avec une Rotation de 180° et un Algorithme Génétique. Ce projet a fait l'object d'une présentation oral. Vous pouvez consulter le code R sur mon dépôt GitHub en cliquant sur le bouton ci-dessous.
